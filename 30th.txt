 >>Using if with .Files.Get (Inside Helm)

Helm allows checking files inside the chart.
Example
{{- if (.Files.Get "files/init.sql") }}
apiVersion: v1
kind: ConfigMap
metadata:
  name: db-init
data:
  init.sql: |
{{ .Files.Get "files/init.sql" | indent 4 }}
{{- end }}
